# Boxphone Revenue Sharing Platform

A multi-product Web3 platform combining **RWA NFTs**, **AI-generated music & streaming**, and an **SMM panel with on-chain escrow**, built on **BNB Smart Chain (BSC)**.

## Technology Stack

- **Blockchain**: BNB Smart Chain 
- **Smart Contracts**: Solidity ^0.8.x  
- **Frontend**: React + web3.js  
- **Backend**: Node.js + Express + MongoDB  
- **AI/Media**: AI music generation pipeline + streaming services  
- **Development**: OpenZeppelin libraries  

## Supported Networks

- **BNB Smart Chain Mainnet** (Chain ID: 56)

## Features

- RWA-backed NFTs for revenue sharing and ownership  
- AI-generated music creation and on-chain-linked streaming  
- On-chain USDT escrow for SMM orders (reserve → capture → release)  
- Admin-controlled pricing and service enable/disable
