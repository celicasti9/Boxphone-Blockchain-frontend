# Public Directory - Logo Placement

## Add Your Logo Here

Place your PHONESTREAM logo file in this directory:

```
frontend/public/logo.png
```

### Current Directory Structure:
```
public/
├── index.html
├── manifest.json
└── logo.png          ← ADD YOUR LOGO FILE HERE
```

## File Requirements

- **Filename**: `logo.png` (or `logo.svg`, `logo.jpg`)
- **Recommended size**: 400x400 pixels or larger
- **Format**: PNG with transparent background (recommended)
- **File size**: Under 500KB for optimal performance

## Quick Steps

1. Copy your PHONESTREAM logo file
2. Paste it into this `public/` directory
3. Name it `logo.png` (or update Header.js if using different name)
4. The logo will automatically appear in the header
